Quilt (2.2)

Phil Stopford

Introduction

Quilt is a production-proven, high performance pattern generator that writes to hierarchical GDSII and Oasis files. The tool also works with the DOE system in Variance (3.5.1 or later). It is available on Windows, macOS and Linux, running on top of .NET Core 3.1

Features

  • Pattern elements based on various shapes (box, L, T, U, X, S, text, bounding box).
    • Text shape for pattern - rectangular primtive that results in a text element in the exported layout.
    • Pattern element from bounding box of pattern.
    • NEW: Arrays can be constructed from the pattern element.
  • Full range of variations for pattern generator: shape characteristics and positioning.
    • Interlayer subshape-relative positioning.
    • IMPROVED: Rotation. including relative rotation. Relative rotation can use reference pivot point.
    • NEW: Array rotation. including relative rotation. Relative rotation can use reference pivot point.
  • Flip (horizontal, vertical, also aligned in either or both of X and Y) support.
  • Text annotation (pins).
  • High performance, multi-threaded pattern generation engine, supporting arbitrary pattern definitions and automatic duplicate removal. 225,000 patterns takes less than 10 seconds.
  • Export to hierarchical GDSII and Oasis. The export process is extremely fast, taking advantage of multithreading and optimized design.
  • Auto-configuration of Variance DOE.
  • Multi-platform : Windows, macOS, Linux. No operational or functional difference across platforms.
    • Modern viewport automatically detects and utilizes optimal framework for platform (Vulkan, OpenGL, Direct3D, Metal).
  • NEW: Moved to .NET Core 3.1 on all platforms. Mono framework is no longer required.

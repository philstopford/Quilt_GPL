Quilt Release History

New to 2.2.1 (TBD)

  • Viewport updates based on upstream project changes.

New to 2.2 (Apr 11 2020)

  • Arrays can be constructed within pattern elements.
    • Each entry in the array comes from the underlying pattern element for the pattern.
    • Array placement can be cross-referenced between elements.
    • The array be be rotated around its bounding box pivot, or that of a reference.
  • Runs on .NET Core 3.1 across all platforms. No longer requires Mono framework.
  • Relative rotations can be performed around the reference element's pivot point.
  • Flipped shapes can now be aligned on either or both of X and Y. Previously, only 'all' was available.

New to 2.1.1 (Dec 30 2019)

  • Support argument to set viewport mode (--graphicsMode x), where x is one of:
    • d3d11 : D3D 11
    • vulkan : Vulkan
    • opengl : OpenGL
    • metal : Metal

New to 2.1 (Nov 20 2019)

  • Bounding box pattern element.
  • Permit negative values across more of the system.
  • Performance tuning : 225k build time on an ultrabook went from 11 seconds to 7.8 seconds with this release.
  • Modern viewport.
    • Faster, and automatically works with the best implementation on each platform (Vulkan, OpenGL, Direct3D11, Metal)
  • Moved to newer .NET framework (4.7.2) and .NET standard 2.0 libraries.

New to 2.0 (Sep 23 2019)

  • Relative positioning in X and Y can now use independent subshape references within the reference element.
  • Rotation support, including relative rotations.
  • Flip support support.
  • Text annotation support.
  • Robustness work.

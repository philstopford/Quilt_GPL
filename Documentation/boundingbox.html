Bounding Box

The bounding box shape defines a rectangle based on the extents of the pattern. The left/right/top/bottom values in this shape define the margin (negative values are permitted). Rotation, flip, etc. are all supported.

Some notes about bounding box elements:

  • Bounding box elements are excluded from the pattern's bounding box evaluation; that is, all bounding box elements in a pattern will start with the same boundary conditions (defined by the min/max extents in X and Y of all other elements in the pattern). This is a design decision to prevent confusion arising from order-of-evaluation (where a second bounding box element in a pattern would otherwise base itself on the extents of the pattern after the first bounding box element is generated).
  • Bounding box elements are not permitted to be used as references by other elements. The bounding box starts as an enclosure of all pattern elements; use as a reference would allow elements to violate that rule, or result in evaluation loops.

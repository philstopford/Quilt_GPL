Text

The text shape defines a rectangle (used to set the contact extents visually) and on export, this rectangle is output along with a text marker in the layout - the intent is to provide text labels for assignment in any parasitic extraction workflow. The text comes from the name of the pattern element. Arrays can be constructed, but be careful that this is suitable for your needs - each text entry in the array will be the same (feedback welcome in case a revision is needed here).

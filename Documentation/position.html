Positioning

Pattern Elements also have many controls for positioning. The initial position for the overall shape is set using the combination of 'subshape reference' and 'position inside subshape'. For shapes involving more than one subshape, the 'subshape reference' drop-down will have more than one entry.

For each of X and Y, you can specify offsets, increments and steps. For patterns with more than one pattern element, you can specify relative positioning using the drop-down.

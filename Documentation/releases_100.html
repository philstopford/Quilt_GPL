New to 1.0.6 (Jun 30 2019)

  • Quilt build further accelerated (multithreading and other optimizations).

New to 1.0.5 (May 30 2019)

  • Option added to allow inputs to be displayed rather than final patterns, aiding usability.
  • Viewport refinements.
  • Documentation available from Help menu.
  • Fixed known bugs (none were serious).

New to 1.0.4 (Apr 30 2019)

  • Small enhancements and fixes to UI, preferences and license activation experience.

New to 1.0.3 (Mar 31 2019)

  • Windows WPF is now primary implementation on that platform.
  • Mac build leaves beta.
    • Performance is comparable to Windows; results are 1:1 matched.
    • Dark mode is supported.
  • Linux/GTK build leaves beta.
    • Performance is comparable to Windows; results are 1:1 matched.
  • UI work continues.

New to 1.0.2 (Feb 26 2019)

  • 70x performance boost due to new duplicate screening approach.
  • UI rework to draw better under GTK.
  • UI performance work through additional event filtering.

New to 1.0.1 (Jan 26 2019)

  • New licensing system.
  • GDSII export improved for readers expecting explicitly closed polygon boundaries.

New to 1.0 (Dec 26 2018)

  • Initial release - everything is new.
  • Writes out hierarchical Oasis/GDSII layout files with factorial matrix of unique patterns arising from the variants defined for the pattern elements.
  • O^2 runtime where O is the increase in patterns within the quilt.
  • Writes out a file that can be used to auto-configure the DOE grid in Variance, and that also contains the definition for each pattern in the grid.
